Core capabilities of the NeuralVault platform.

---

## Semantic Search

Search by meaning instead of exact words.

The system understands intent and retrieves contextually relevant information even when the wording doesn't match exactly.

---

## AI Memory

The AI remembers:

- Projects
- Documentation
- Notes
- Architecture decisions
- Previous fixes

Information persists across sessions via the vector database.

---

## Multi-source Context

The same chat session can retrieve data from:

- Multiple Obsidian vaults
- Multiple repositories
- Multiple documentation sources

All merged into a single, coherent context window.

---

## Context Optimization

The system:

- Filters irrelevant chunks
- Compresses context
- Reduces token usage
- Improves retrieval accuracy

This keeps AI responses focused and cost-efficient.

---

## Multi-model Support

Users can:

- Choose the LLM provider manually
- Use automatic routing
- Use local models (Ollama)
- Use cloud models (OpenAI, Claude, Gemini, etc.)

---

## BYOK — Bring Your Own Key

Users can add their own API keys:

- OpenAI
- Anthropic
- Google Gemini
- OpenRouter

The SaaS monetizes **memory**, **retrieval**, **orchestration**, and **contextual intelligence** — not token reselling.
